# 🤖 Machine Learning - Learning Repository

A personal learning repository following the 
[Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) 
taught by **Andrew Ng**, offered by DeepLearning.AI and Stanford University on Coursera.

---

## 📚 Source & Reference

> Ng, Andrew. *Machine Learning Specialization*.
> DeepLearning.AI & Stanford University via Coursera.
> 🔗 https://www.coursera.org/learn/machine-learning

All code and concepts in this repository are based on the course lectures and labs.
This repo is for personal learning purposes only.

## 👨‍🏫 About the Instructor

**Andrew Ng** is a globally recognized AI researcher and educator. He is:
- Co-founder of Coursera
- Former head of Google Brain
- Former Chief Scientist at Baidu
- Faculty at Stanford University
- Founder of DeepLearning.AI

His Machine Learning course has been taken by over **4.8 million learners** worldwide since 2012.

## 📖 Key Concepts Covered

**Week 1 - Introduction to Machine Learning**
- Supervised vs unsupervised learning
- Linear regression model
- Cost function & gradient descent

**Week 2 - Regression with Multiple Input Variables**
- Multiple linear regression & vectorization
- Feature scaling and learning rate tuning
- Feature engineering
- Polynomial regression

**Week 3 - Classification**
- Logistic regression & decision boundaries
- Cost function for logistic regression
- Overfitting and regularization
